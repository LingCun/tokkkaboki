## 파일 구조
```
index.html            홈. 업로드→인원감지→모드추천→결과페이지 이동(sessionStorage 전달)
guide.html            카톡 내보내기 가이드 (PC/iOS/Android 탭, UA 자동선택)
analyze.html          빌런 리포트 결과 UI
breakup-forecast.html 연인 예보 결과 UI
parser.js             카톡 .txt 파서 (PC/Android/iOS + 멀티라인). KakaoParser.{parseKakao,sample}
stats.js              빌런 분석. KakaoStats.{analyze,select} — 상 카탈로그 18종
breakup-stats.js      연인 분석. BreakupStats.{analyze,flavor,sample} — 애정온도 복합점수
ad.js                 분석중 광고 게이트. AdGate.run(sec,title) — 카카오 애드핏
share.js              결과 카드 PNG 생성+공유. Share.{villain,breakup} — Canvas 직접 그림
GUIDE.md              내보내기 가이드 문서
```

## 동작 흐름
- **진입(샘플)**: 페이지 들어오면 샘플을 **"예시(광고)" 모드**로 자동 표시 (코너 "예시" 리본 + 빌런은 데모CTA). 광고 게이트 없음.
- **내 결과(mine)**: .txt 업로드/붙여넣기 → **분석중 광고 10초** → "결과 보기" → 렌더 + **완료 팝업** → 닫으면 결과 카드로 스크롤+포커스. "내 단톡 결과"/"우리 관계 예보" 배지로 예시와 구분.
- **다양성**: 빌런=🎲 다시뽑기(센 상 2 고정 + 가중랜덤 로테이션, 1인1관왕). 연인=🎲 다른 코멘트(온도 고정, 페르소나·문구만 변형).
- **공유**: 📤 결과 공유 → Canvas로 카드 PNG → `navigator.share`(모바일 이미지 첨부) / 폴백 PNG저장+캡션복사.
